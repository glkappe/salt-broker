=====================
What is Salt Broker?
=====================

salt-broker is a lightweight proxy for `saltstack <https://github.com/saltstack/salt>`_


How to use it?
=====================

Install zeroMQ
-----------------

Config salt-broker
-------------------

change the 'master' address in 'salt-broker.py'

Start salt-broker
-------------------

.. code-block:: bash

    python salt-broker.py
